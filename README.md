# Cars Dealership Full-Stack Capstone Project

## Repository Name
dealer_evaluation_backend2

## Project Name
Cars Dealership Review Web Application

## Project Description
This project is a full-stack web application developed for **Cars Dealership**, a national car retailer in the United States. The system allows users to browse dealership branches, view customer reviews, and submit new reviews through an interactive and responsive web interface.

The application demonstrates full-stack development skills including frontend development, backend API integration, database management, and deployment readiness.

## Features
- View dealership locations across the U.S.
- Read customer reviews for each dealership
- Submit new reviews
- Responsive web design for desktop and mobile
- Backend API for dealership and review data

## Technologies Used
- Python / Django
- HTML / CSS / JavaScript
- Bootstrap
- REST APIs
- Database Management System
